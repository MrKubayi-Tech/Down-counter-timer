# Countdown Timer ⏳

A simple, responsive countdown timer built with **HTML**, **CSS**, and **JavaScript**.  
It lets users set a custom time (HH:MM:SS), start the countdown, stop it midway, and automatically reset to the default template once completed.

---

## 🚀 Features
- **Set Timer:** Input hours, minutes, and seconds manually.
- **Start Countdown:** Begins decrementing until zero.
- **Completion Message:** Displays a “Timer Complete” message when finished.
- **Stop Button:** Interrupts the countdown and resets to default (HH:MM:SS).
- **Responsive Design:** Works smoothly across browsers and screen sizes.

---

## 🧠 How It Works
1. Enter your desired time in the input field.
2. Click **Start** to begin the countdown.
3. Click **Stop** anytime to reset.
4. When the timer reaches zero, a completion message appears and the display resets.

---

## 🛠️ Technologies Used
- **HTML5** – Structure and layout  
- **CSS3** – Styling and animations  
- **JavaScript (ES6)** – Logic and interactivity
